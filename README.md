
# 🎯 Number Guessing Game (Streamlit)

An interactive web-based number guessing game built using Python and Streamlit. The application generates a random number and allows users to guess it with real-time feedback.

---

## 🚀 Live Demo

👉 [https://your-app-name.streamlit.app](https://number-guessing-game-nuytxvbznlzdrwvwtz3qb2.streamlit.app/)

---

## 🧠 Features

* 🎲 Random number generation (1–100)
* 📊 Tracks number of attempts
* ⚡ Real-time feedback (Too High / Too Low / Correct)
* 🔄 Restart game functionality
* 🌐 Interactive web interface using Streamlit

---

## 🛠️ Technologies Used

* Python
* Streamlit

---

## 📦 Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/yourusername/number-guessing-game.git
```

2. Navigate to project folder:

```bash
cd number-guessing-game
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
streamlit run guessing_random_numbers.py
```
